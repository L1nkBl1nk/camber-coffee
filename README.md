# Camber Coffee ☕

A warm, artisan **landing page for a specialty coffee shop**, built as a portfolio
piece to showcase clean, fully-responsive front-end work — no frameworks, no build step.

🔗 **[Live demo →](https://camber-coffee.vercel.app)**

![Camber Coffee preview](https://image.thum.io/get/width/1200/https://camber-coffee.vercel.app)

## The brief
A small specialty roaster needs a site that feels premium and inviting, shows the
menu, and turns visitors into walk-in customers — fast to load and great on mobile.

## What I built
- **Sticky header** that fades in a background on scroll
- **Hero** with a floating product image and an animated rotating seal
- **Tabbed menu** (Coffee / Tea / Desserts) rendered from a small JS data model
- **Masonry gallery**, customer **reviews**, opening hours and an embedded map
- **Scroll-reveal** animations, **mobile burger menu**, and a newsletter form
- Fully responsive (desktop / tablet / mobile) with reduced-motion support

## Tech
- **HTML5**, **CSS3** (custom properties, grid & flexbox)
- **Vanilla JavaScript** — tabs, scroll reveal, mobile nav, form handling
- Google Fonts (Cormorant Garamond + Manrope) · no build tooling

## Run locally
It's a static site — just open `index.html` in your browser.

```bash
git clone https://github.com/L1nkBl1nk/camber-coffee.git
cd camber-coffee
# then open index.html
```
